# CROSfetch
My clone of neofetch, made for chrome os (github/dylanaraps/neofetch).  
Mostly to learn more bash.  

## Info Given:
 - Basic prompt (As a title)
 - Username
 - Google accounts on 
 - Hostname
 - OS
 - Kernel
 - Uptime
 - Package managers
 - Packages
 - Terminal
 - Shell
 - Your PATH ($PATH)
 - DE
 - Theme 
 - WM
 - CPU
 - GPU
 - Memory
 - Drive (Boot drive)
