A Neofetch clone built from the ground up for Chrome OS  

 
## Info Given:
 - Basic prompt (As a title)
 - Google accounts on Device
 - OS
 - Channel
 - Kerne1
 - Host
 - Uptime
 - Package managers
 - Packages
 - Terminal
 - Shell
 - Program Locations (Your ```$PATH```)
 - DE
 - WM
 - Theme
 - CPU
 - GPU
 - Memory Amount
 - Storage Amount
